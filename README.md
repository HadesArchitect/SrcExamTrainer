# SRC Exam Trainer

A React Native Expo app for training SRC radio examination.

## Setup

```bash
npm install
```

## Launch in Emulator

### Web (Recommended)
```bash
npm run web
```

### Android
Requires Android emulator

```bash
npm run android
```

### iOS
Requires Xcode

```bash
npm run ios
```

### Development Server
```bash
npm start
```

## Testing
```bash
npm test
```

## Requirements
- Node.js
- Expo CLI
